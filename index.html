Instrucciones

Un proceso LEGO para crear activos digitales

Primero se ordena la idea. Despues se orienta al buyer persona. Por ultimo, el prompt convierte esa estructura en una pagina web con visor, codigo editable y valor reutilizable.

1

Ordenar la idea

Define que conocimiento, practica, servicio o propuesta quieres convertir en recurso util.

2

Crear la ficha

Genera el buyer persona para que el activo responda a un perfil concreto y no a una audiencia abstracta.

3

Generar el activo

Copia el prompt, adjunta ficha y logo, y construye una primera pagina con visor y codigo editable.

Prompt definitivo

La pieza que convierte la idea en pagina web

Este prompt obliga a que la web nazca de una ficha de buyer persona. Si no hay ficha, la IA debe detenerse y pedirla. Asi el resultado deja de ser un diseno generico y se convierte en un activo digital con foco, estructura y utilidad.

Regla Doc ROI

Un activo digital no es solo un archivo bonito. Es conocimiento ordenado, orientado a un usuario, empaquetado para ser usado, compartido, aprendido y mejorado.
